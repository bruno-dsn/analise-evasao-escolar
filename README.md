# Análise da Evasão Escolar no Ensino Médio Brasileiro

Este projeto tem como objetivo aplicar conceitos iniciais de análise de dados
utilizando dados públicos para compreender padrões de evasão escolar no ensino
médio no Brasil.

A proposta do projeto é demonstrar como a análise de dados pode apoiar a
compreensão de problemas sociais reais e auxiliar na tomada de decisão baseada
em evidências.

## Objetivo
Analisar dados educacionais para identificar:
- Regiões com maiores taxas de evasão escolar
- Evolução da evasão ao longo do tempo
- Possíveis relações entre fatores socioeconômicos e evasão

## Ferramentas Utilizadas
- Python (Pandas, Matplotlib)
- Google Colab
- Google Planilhas
- GitHub
- Looker Studio

## Estrutura do Projeto
- `data/`: dados brutos e tratados
