## Fontes de Dados Utilizadas

### INEP – Censo Escolar
- Descrição: dados oficiais sobre matrículas, aprovação, reprovação e abandono
escolar no Brasil.
- Tipo de dados: estruturados (CSV).
- Forma de acesso: download direto no portal do INEP.

### IBGE – Indicadores Socioeconômicos
- Descrição: dados sobre renda média e características regionais.
- Tipo de dados: estruturados.
- Forma de acesso: download pelo site do IBGE.

## Observação
Todos os dados utilizados são públicos e não confidenciais.
